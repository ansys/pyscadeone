PyScadeOne is a Python library for reading 
`Ansys Scade One <https://www.ansys.com/products/embedded-software/ansys-scade-one>`_:superscript:`TM`  
models and providing various services for these models.

PyScadeOne is tested with Python 3.9 and 3.12.
